ChronoWhoop

06 Armed

Flying — read across the room

Extreme size contrast for a glance from the pilot's seat: the running lap clock is huge and white for maximum legibility, with last lap, session best (amber), and lap count below. The pulsing cyan dot means the sensor is live; the wake-lock chip means the screen won't sleep. STOP and Discard last lap are thumb-sized.

08 Edge state

Unsupported browser

A per-capability probe instead of a dead end: WebGPU, camera, OPFS, and Web Speech each report pass or fail, so the missing piece is obvious. Here WebGPU is unavailable; guidance points to a supported Chromium build.
